About me

嗨，我是 Zanya。 我喜歡研究各種系統，從 Web 漏洞、Pwn 題目到網路封包，看看它們是怎麼運作、又會在哪裡出錯。 CTF / 靶機 對我來說，不只是比賽，而是一種把原理轉化為實踐的方法。每一道題目，就像是一個小黑盒，等著被拆解與理解。 這裡會記錄我的學習筆記、比賽複盤，以及一些關於學習方法的想法。

接觸領域

Web 漏洞分析

研究常見 Web 安全漏洞，如 SQL Injection, XSS, CSRF 等，並探索其背後的攻防原理。

Pwn & Binary Exploitation

挑戰 CTF 中的 Pwn 題目，學習記憶體管理、Shellcode 編寫與常見的二進位漏洞利用技術。

網路封包分析

使用 Wireshark 等工具分析網路流量，從封包層級理解協定與應用程式的行為。

滲透測試

模擬真實世界的攻擊者，採用系統性的方法論，從初期偵察、漏洞利用，到橫向移動與權限維持，找出潛在的攻擊路徑，並評估整體的安全防禦體系。
